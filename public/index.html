DEPLOYMENT

Up and Running in
3 Steps.

Fork, pair, and deploy. No experience required.

01

Fork the Repository

Fork the NEXUS-MD repo to your GitHub account. One click gets you the full bot codebase.

Fork on GitHub
02

Get Your Session ID

Visit the NEXUS pairing terminal to link your WhatsApp number and generate a session ID.

Pair Code QR Code
03

Deploy & Go Live

Deploy to Heroku, a VPS, or any panel. Set SESSION_ID and ADMIN_NUMBERS in config vars and you're live.

Deploy to Heroku https://verification-rho-ten.vercel.app/

Running a panel (Pterodactyl / cPanel / VPS)?

bash 
# Download ZIP, upload & extract, then run:
bash scripts/panel.sh

The script auto-installs dependencies and starts the bot. No extra setup needed.

COMMANDS

Key Commands
at a Glance.

Send .menu in any WhatsApp chat to see all 50+ commands.

.ai [text] Chat with AI
.menu Full command list
.ping Bot status & latency
.sticker Media → sticker
.yt [url] YouTube audio download
.epl Live EPL scores
.tagall Tag all group members
.antidelete on Recover deleted msgs
.alive Check bot is running
.tts [text] Text to speech
.kick @user Remove from group
.weather [city] Weather forecast
CONFIGURATION

Environment
Variables.

Variable Description Required
SESSION_ID Your WhatsApp session from the pairing terminal Required
ADMIN_NUMBERS Your number e.g. 254706535581 Required
PORT Dashboard port (default: 5000) Optional
DATABASE_URL PostgreSQL URL — local file used if blank Optional
